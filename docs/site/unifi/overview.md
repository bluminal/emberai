The **unifi** plugin provides intelligent network operations for UniFi networks through Claude. It covers device topology, wireless health, client management, traffic analysis, security posture, and multi-site operations.

## Capabilities

- **Topology discovery** — scan all devices, uplinks, and VLANs
- **Health monitoring** — device status, firmware currency, ISP metrics, event logs
- **Client management** — find, diagnose, and track any connected device
- **Commands** — high-level operational commands (`unifi scan`, `unifi health`)
